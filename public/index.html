GoldenCheetah Proxy Demo by SweatStack

Browse your GoldenCheetah data from the browser. By SweatStack.


This app connects to GoldenCheetah running on your computer. Because your browser can't talk directly to GoldenCheetah, we use a secure tunnel to create a temporary link. Nothing is stored on our servers.

  1. Open GoldenCheetah and enable the API:
    Settings → General → Integration → Enable API Web Services
  2. Open a terminal (Mac: Terminal app, Windows: PowerShell) and run this command:
    ssh -p 443 -R0:localhost:12021 a.pinggy.io
    This uses SSH, which is already installed on your computer. No extra software needed.
  3. The terminal will show a URL like https://xxxx.a.pinggy.io. Paste it below and press Enter.
